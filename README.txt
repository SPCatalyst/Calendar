== Description ==

The catalyst calendar can be implemented in your WordPress  website.

Use the shortcode [community_calendar] to load it on any page.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `spc-community-calendar.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates

== Frequently Asked Questions ==

= Can i load only our events? =

Sure. Go to settings and select the "Internal" option in the type setting.
